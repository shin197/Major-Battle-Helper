# 코코포리아 매니저 (Major-Battle-Helper)

코코포리아 매니저는 온라인 TRPG 플랫폼인 **코코포리아(CCfolia)**의 사용성을 극대화하기 위해 개발된 Chrome 확장 프로그램입니다. 
코코포리아의 내부 상태(Redux)와 Webpack 모듈을 탈취(Hijacking)하여 동작하는 강력한 범용 API를 기반으로, 다양한 **편의성(QoL) 기능**과 특정 룰을 위한 **전용 전투 시스템(Major Battle)**을 제공합니다.

---

## 🛠 아키텍처 및 특징 (World Isolation)

본 확장 프로그램은 크롬 확장 프로그램의 보안 한계를 극복하기 위해 두 가지 실행 환경(World)을 브릿지로 연결하여 동작합니다.

1. **MAIN World (`/contents/inject/`)**:
   - 코코포리아 원본 JavaScript와 동일한 메모리 공간에서 실행됩니다.
   - `hijack.ts`를 통해 React Fiber 트리, Redux Store, Firebase DB 인스턴스 권한을 획득합니다.
   - 토큰 생성/조작, 캐릭터 수정 등을 수행하는 내부 API(`ccfoliaAPI`)를 구축합니다.
2. **ISOLATED World (`/contents/`, `/core/`)**:
   - 확장 프로그램의 독립된 메모리 공간에서 실행되며, 안전하게 DOM 요소에 접근하여 UI를 추가합니다 (MutationObserver 등).
   - `ccf` 객체를 통해 `postMessage` 기반의 비동기 RPC 통신으로 MAIN World의 기능을 원격 호출합니다.

## 🚀 주요 기능 및 사용법

확장 프로그램을 설치하고 팝업창을 열면 각 기능을 토글 형태로 활성화/비활성화할 수 있습니다. 

### 1. 범용 편의성 기능 (General QoL)

룰에 상관없이 모든 코코포리아 방에서 사용할 수 있는 편의 기능들입니다.

*   **다중 토큰 제어 (Bulk Drag)**
    *   `마우스 휠 클릭 + 드래그`: 여러 토큰을 한 번에 박스 형태로 선택합니다.
    *   `Ctrl + 휠 드래그`: 기존 선택 범위에 토큰을 추가합니다.
    *   `Shift + 휠 드래그`: 선택 범위에서 제외합니다.
    *   `Ctrl + C` / `Delete`: 다중 선택된 토큰들을 한 번에 복사하거나 삭제할 수 있습니다. (현재 스크린 패널 및 마커 패널에 적용)
*   **향상된 마우스 액션 (Enhanced Mouse Action)**
    *   `Ctrl + 좌클릭`: 클릭한 위치에 핑(Ping)을 찍습니다.
    *   `Alt + 좌클릭`: 캐릭터 토큰을 클릭하면 채팅창에 해당 캐릭터의 이름이 `[캐릭터]` 형식으로 자동 입력됩니다.
    *   `E (토큰 위에서)`: 토큰 이미지를 팝업으로 크게 확대해서 보여줍니다.
*   **커스텀 클립보드**
    *   스크린 패널 및 마커 패널을 클립보드에서 바로 붙여넣기(Paste)할 수 있도록 지원합니다.
*   **매크로 슬롯 단축키 (Slot Shortcuts)**
    *   활성화된 캐릭터를 빠르게 전환할 수 있습니다. (채팅창 활성화 상태에서 사용)
    *   `Ctrl + 숫자키(1~0)`: 현재 선택한 캐릭터를 해당 슬롯에 저장합니다.
    *   `Alt + 숫자키(1~0)`: 저장된 슬롯의 캐릭터로 즉시 전환합니다.
*   **채팅 로그 페이저 (Log Pager)**
    *   채팅 로그 창 하단에 페이지 넘기기(Pagination) 버튼을 추가하여 수많은 과거 로그들을 쉽게 탐색할 수 있습니다.
*   **채팅창 기능 향상 (Chat Input)**
    *   **수식 자동 계산**: 채팅 입력 시 `Ctrl + Enter`를 누르면 텍스트 내 포함된 수식을 계산합니다. (`+`, `-`, `*`, `/`, `\`, `%` 지원)
    *   **변수 참조**: `{변수.max}` 등을 사용하여 스테이터스의 최대값을 불러올 수 있습니다.
    *   **히스토리 탐색**: 키보드 `위/아래 방향키`를 통해 이전에 전송했던 채팅을 다시 불러옵니다.
    *   **자동 완성**: 괄호 등 기호를 자동으로 완성해줍니다.
    *   **시스템 메시지**: `/g` 또는 `/ㅎ` 명령어 뒤에 메시지를 입력하면 코코포리아 시스템(GM) 메시지로 전송됩니다.
    *   **빠른 메시지 삭제**: 채팅 메시지 위에 마우스를 올리면 삭제 버튼이 표시되어 빠르게 삭제 가능합니다.
    *   **스테이터스 빠른 변경**: `/stat [캐릭터명] 스테이터스이름(=, +, -)수치` 형태로 입력하여 능력치를 바로 수정할 수 있습니다.
    *   **상한 제한 적용**: `/cap` 명령어를 사용하면 활성화된 캐릭터들의 스테이터스를 최대치 이내로 맞춰줍니다.
*   **표정 일괄 복사 및 변경 (Face Button & Copy Faces)**
    *   캐릭터 컨텍스트 메뉴에 "표정 복사" 항목을 주입하여 클립보드에 표정 JSON을 한 번에 추출합니다.
    *   화면에 캐릭터의 표정을 즉각적으로 바꿀 수 있는 UI 버튼을 추가합니다.
*   **홈 방문 기록 (Home History)**
    *   코코포리아 홈(`/home`) 페이지 진입 시 최근 방문했던 방 목록을 표시해줍니다.
*   **AI NPC 보조 (AI Chat)**
    *   확장 프로그램 팝업 설정에서 OpenAI API Key와 System Prompt를 등록하면, 채팅 입력창에서 AI가 작성한 NPC 대사를 제안받을 수 있습니다.

### 2. 특정 룰 전용 기능 (Major Battle Mode)

특정 전투 룰이나 시스템에 특화된 기능을 제공합니다. 환경 변수(`PLASMO_PUBLIC_MAJOR_BATTLE`) 설정에 따라 활성화 여부가 결정됩니다.

*   **동적 변수 치환**
    *   `{S}`: 바로 직전에 굴린 주사위 판정의 **성공도**를 가져옵니다.
    *   `{S.#}`: 바로 직전에 굴린 주사위 판정의 **성공한 개체수**를 가져옵니다.
*   **전투 초기화 기능**
    *   `/battle` 명령어를 채팅에 치면 전투 시작 시 필요한 맵 세팅, 캐릭터 배치, 상태 이상 초기화 등을 자동화합니다.
*   **자동 대미지 처리**
    *   `/dmg [캐릭터명] 피해량 x횟수 타입` 형태의 명령어 입력을 통해 특정 캐릭터에게 대미지를 가합니다. (피해 타입 예시: 충격, 관통, 방사 등)
    *   내부적으로 `dice-result.ts` 등을 통해 주사위 결과를 감지, 대미지 계산 및 상태 이상 적용을 자동화할 수 있습니다.
*   **유닛 수 조절**
    *   `/u [캐릭터명] 유닛수` 명령어로 특정 캐릭터의 유닛 수를 설정할 수 있습니다. (유닛 수 앞에 `+` 또는 `-` 를 붙이면 기존 수치에서 증감 처리됩니다.)

## 💻 개발 및 빌드 (Development)

이 프로젝트는 [Plasmo](https://docs.plasmo.com/) 프레임워크를 기반으로 구성되었습니다.

**개발 서버 실행**
```bash
pnpm dev
# 또는 npm run dev
```

**프로덕션 빌드 (스토어 배포용)**
```bash
pnpm build
# 또는 npm run build
```

코드를 수정하면 팝업(`popup.tsx`)이나 백그라운드 스크립트, 컨텐츠 스크립트 들이 핫 리로드(Hot Reload) 또는 자동 업데이트 됩니다. 특정 DOM 요소가 로딩되기를 기다리거나, 리액트 파이버를 통해 토큰 정보를 추출하는 등 코어 기능들은 `/utils/` 폴더 내 `wait-for.ts`, `token.ts` 등을 참고하세요.
